
_Литерал_ - последовательность символов в коде программы, которая обозначает значение некоторого конкретного типа.

```cpp
5;
0.2;
"abc";
'a';
```

_Литеральный суффикс_
```cpp
0.2f; // меняет тип литерала: double -> float
5u;  // int -> unsigned int
```

```cpp
int x = 5u;  // неявное преобразование unsigned int -> int
// такие преобразования опасны возможным переполнением
```

_interger promotions_
```cpp
char -> int -> long -> long long
```

_floating point promotions_
```cpp
float -> double
```


Неявные конверсии - зло, их следует избегать.
```cpp
0 - 1u; // результат - unsigned int, равный 4 млрд
```

Запись в 16ричной системе счисления
```cpp
0xff;  // 256
```

в 8ричной
```cpp
0123;  // 83
```

в двоичной (c C++14)
```cpp
0b101;
```
